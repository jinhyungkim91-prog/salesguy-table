# 비즈니스 스코어 정책

> 세일즈가이의 식탁 — 비즈니스 탭 식당 점수 산정 기준  
> 최대 100점 (내부 합산 최대 107 → cap 100) / `src/App.js > calcBizScore()` 함수 기준

---

## 점수 구성 (5개 항목 + 보너스)

| 항목 | 최대 | 설명 |
|---|---|---|
| ① 룸 상태 | 35점 | 프라이버시 확보 수준 |
| ② 장르 격식 | 25점 | 비즈니스 식사 적합도 |
| ③ 평점 | 15점 | 네이버·구글 등 실사용 평점 |
| ④ 출처 신뢰도 | 15점 | 미슐랭·블루리본 등 공신력 |
| ⑤ 보너스 | 최대 12점 | 특별 시설·등급 가산 |
| ⑥ 리뷰수 신뢰도 | 최대 5점 | 네이버 방문자 리뷰수 기준 |

---

## ① 룸 상태 (최대 35점)

| 조건 | 점수 | 판단 근거 |
|---|---|---|
| 전석 룸 / 전층 룸 | 35점 | note에 "전석" 또는 "전층" 포함 |
| 개별룸 / 개인실 / 프라이빗룸 | 28점 | note에 해당 키워드 포함 |
| 룸 있음 (확인) | 25점 | room 필드 = `✅ 확인` |
| 조건부 룸 | 5점 | note에 "조건" 포함 |
| 룸 없음 | 0점 | 해당 없음 |

---

## ② 장르 격식 (최대 25점)

| 장르 | 점수 | 예시 |
|---|---|---|
| 파인다이닝 / 오마카세 | 25점 | 미슐랭 레스토랑, 스시 오마카세 |
| 한정식 / 일식코스 / 코스 / 프렌치 / 이탈리안 | 20점 | 한정식 코스, 프렌치 파인다이닝 |
| 한우 / 해산물 / 회 / BBQ / 갈비 / 스테이크 | 15점 | 한우 구이, 랍스터 등 |
| 중식 / 양식 / 샤브샤브 / 훠궈 / 딤섬 | 10점 | 중식당, 딤섬 레스토랑 |
| 기타 (한식, 고기구이 등) | 5점 | 일반 한식, 분식 등 |

---

## ③ 평점 (최대 15점)

| 평점 | 점수 |
|---|---|
| 4.7 이상 | 15점 |
| 4.5 ~ 4.69 | 12점 |
| 4.3 ~ 4.49 | 9점 |
| 4.0 ~ 4.29 | 6점 |
| 4.0 미만 또는 없음 | 3점 |

> 평점 출처: 네이버, 구글, 카카오맵 등 플랫폼 평점 수동 입력

---

## ④ 출처 신뢰도 (최대 15점)

| 출처 | 점수 |
|---|---|
| 미슐랭 가이드 | 15점 |
| 블루리본 2025·2026 | 13점 |
| 블루리본 (연도 미지정) | 11점 |
| 서울미식100 / 준성기 | 9점 |
| 다이닝코드 / 구글 / 캐치테이블 / 식신 / 네이버 | 7점 |
| 기타 (직접 확인 등) | 4점 |

> source 필드에 출처 키워드가 포함되어야 자동 인식됨

---

## ⑤ 보너스 (최대 12점)

특별 시설·등급에 대한 추가 가산점. 아래 조건 중 해당하는 최고점 1개 적용.

| 조건 | 점수 | 판단 근거 |
|---|---|---|
| 미슐랭 3스타 | 12점 | source 또는 note에 "미슐랭3스타" / "3스타" |
| 미슐랭 2스타 | 10점 | source 또는 note에 "미슐랭2스타" / "2스타" |
| 미슐랭 1스타 (기타) | 9점 | source에 "미슐랭" 포함 |
| 발렛 + 전담/VIP/소믈리에 | 10점 | 복합 조건 동시 충족 |
| 발렛 / VIP룸 | 8점 | 해당 키워드 포함 |
| 상견례 / 콜키지 / 1++ / BMS9 | 8점 | 고급 접대 키워드 |
| 프라이빗 / 미팅룸 / 비즈니스 / 접대 | 6점 | 비즈니스 특화 키워드 |
| 전담 / 소믈리에 / 럭셔리 | 5점 | 서비스 품질 키워드 |
| 룸 / 개인실 (일반) | 4점 | 기본 룸 키워드 |
| 기타 | 2점 | 기본값 |

---

## ⑥ 리뷰수 신뢰도 (최대 5점)

네이버 방문자 리뷰수 기준. `reviews` 필드에 숫자로 입력.

| 리뷰수 | 점수 |
|---|---|
| 1,000 이상 | 5점 |
| 500 ~ 999 | 3점 |
| 200 ~ 499 | 2점 |
| 100 ~ 199 | 1점 |
| 100 미만 또는 없음 | 0점 |

> 리뷰수는 `restaurants_rating_review.csv`에서 일괄 입력 후 반영

---

## 점수 해석 기준 (참고)

| 점수 | 등급 | 의미 |
|---|---|---|
| 90점 이상 | ⭐⭐⭐ | 최상급 접대 — 미슐랭·전석룸·오마카세 |
| 80~89점 | ⭐⭐ | 고급 비즈니스 식사 — 한정식·한우·개별룸 |
| 70~79점 | ⭐ | 실용적 비즈니스 식사 — 룸 있는 중식·일식 |
| 70점 미만 | — | 일반 식사 (룸 없음 또는 평점 낮음) |

---

## 수동 score 입력 정책

- `calcBizScore()`는 **자동 계산용** 함수로, 실제 데이터의 `score` 필드는 **수동 입력값 우선**
- 자동 계산 결과와 실제 경험이 다를 경우 JSON에서 `score` 직접 수정
- 미등록 식당 신규 추가 시 장르·평점 기준으로 **기본 score 75~82** 자동 부여
